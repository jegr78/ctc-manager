Tips
  • Graphic size is 1920×1080px (Full HD 16:9) — matches the screenshot viewport.
  • Defaults are plain text; match-specific values use th:text="${v.key} ?: 'Default'".
  • To make a setting match-specific, wrap its value with that th:text; to pin it, remove the attribute.
  • Missing v.* keys fall back to the ?: default (no template error).
