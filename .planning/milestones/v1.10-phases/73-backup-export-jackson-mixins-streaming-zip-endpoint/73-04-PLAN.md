## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client ŌåÆ controller | Any HTTP client (browser, curl, Playwright) ŌåÆ `BackupController` (admin endpoint). Trust drops at controller entry; auth + CSRF are mandatory on prod/docker. |
| controller ŌåÆ archive service | `BackupController.export()` constructs an `Instant` and delegates to `BackupArchiveService.writeZip(OutputStream, Instant)` inside a `StreamingResponseBody` lambda. No user input crosses this boundary. |
| controller ŌåÆ response stream | `StreamingResponseBody` runs OUTSIDE the controller thread; mid-stream exceptions can produce a truncated ZIP per RESEARCH L-10. |

| T-73-02 | Tampering | Missing or forged CSRF token enables CSRF-driven backup exfiltration (attacker tricks an authenticated admin's browser into firing a cross-origin POST to /admin/backup/export) | mitigate | Spring Security 7 CSRF on by default on prod/docker (`SecurityConfig` does NOT call `.csrf(...)`).
