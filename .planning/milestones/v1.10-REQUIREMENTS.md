## Out of Scope (Documented)

| Feature | Reason |
| ------- | ------ |
| Cloud-Backup-Destinations (S3/GCS/Azure) | Single-Admin-Liga, lokales File-System ausreichend |
| Scheduled / Cron'd Backups | Manueller Button-only ist explizite User-Wahl |
| Multi-Format-Support (CSV/XML/SQL) | JSON ist für Round-Trip ausreichend |
| Merge-Import-Conflict-Policy | Replace-All ist sicherer (User-Choice 2026-05-09) |
| Encryption at Rest | PSN-IDs sind öffentliche IDs; deferred |
| Async Export mit Progress-Polling | StreamingResponseBody mit ≤100 MB genügt |
| Maven Enforcer Custom Rule | `exec-maven-plugin`-grep-gate (PLAT-07) genügt |
| Forward-/Backward-Compat-Migration zwischen Schema-Versionen | Schema-Version-Strict-Equality genügt für MVP |
