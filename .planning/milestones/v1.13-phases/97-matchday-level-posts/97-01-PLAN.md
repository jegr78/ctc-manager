<objective>
Plan 97-01 ships POST-06 — the per-match Match Preview Announcement button on
`/admin/matches/{id}` and its auto-edit hook on `streamLink` / `discordTeaser`
changes — closing the first of three Phase 97 post-type matrices.

Purpose: Operator can announce a match to the announcement channel with one
click, attaching the Settings + Lineups graphics with a structured Markdown
header. Later edits to streamLink or teaser automatically PATCH the existing
Discord message so the announcement always reflects current data.

Output: One new event record, one new pre-flight DTO record, three modified
service classes, one modified controller, one modified template, two
artifact revisions (UI-SPEC + REQUIREMENTS), and five new test classes.
</objective>
