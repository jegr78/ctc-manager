## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-67-04 | Information Disclosure | Templates rendered to public HTML | mitigate (net-positive) | Removing internal planning-artifact references (`D-XX`, `Plan 60-NN`, `Pitfall N`) from comments that previously leaked to browser HTML output reduces the project's information-disclosure surface. The removed strings reference internal planning artifacts only — no secrets, credentials, or PII. The `.planning/` directory is committed to git, so the underlying decisions remain public — we are removing duplication, not data. |

| T-67-06 | Other STRIDE | — | accept | No identity, runtime, or authorization code is touched. |
</threat_model>
