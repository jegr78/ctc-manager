## Interaction Contract

Phase 37 fixes link resolution. The interaction model is unchanged — all elements remain passive anchors. No new interactive patterns are introduced.

| Element | Interaction | State | Behavior |
|---------|-------------|-------|----------|
| Archive season links | click | default | Navigate to `season/{slug}/standings.html` |
| Nav "Driver Ranking" | click | default | Navigate to `season/{activeSeasonSlug}/driver-ranking.html` |
| Nav "Driver Ranking" | — | hidden | Not rendered when `activeSeasonSlug` is empty |
| Root-level nav links | click | default | Use `./` prefix instead of `/` prefix |
| Team logo `<img>` | none | default | Displays from `assets/img/logos/teams/{uuid}/{filename}` |
| Team logo `<img>` | none | missing | Not rendered when team has no logo (existing `th:if` guard) |
