**Part A: Create `src/main/resources/templates/site/teams.html`** (per D-02, D-09, D-10, D-11, D-12, D-13, D-17, D-18):

```html
<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org"
      th:replace="~{site/layout :: layout('Teams', ~{::section})}">
<body>
<section>
    <div class="section">
        <h2 class="section-title">Teams</h2>
        <div class="filter-bar">
            <select id="season-filter">
                <option value="">All Seasons</option>
                <option th:each="entry : ${seasonEntries}"
                        th:value="${entry.slug()}"
                        th:text="${entry.season().displayLabel}">Season</option>
            </select>
        </div>
        <div class="overview-grid">
            <div th:each="entry : ${teamEntries}"
                 class="overview-card"
                 th:attr="data-seasons=${#strings.listJoin(entry.seasonSlugs(), ' ')}">
                <img th:if="${entry.logoRelPath() != null}"
                     th:src="${entry.logoRelPath()}"
                     class="overview-card-logo"
                     th:alt="${entry.shortName()}"/>
                <a th:href="${entry.profileUrl()}"
                   th:text="${entry.shortName()}"
                   class="overview-card-name"></a>
                <div class="overview-card-seasons">
                    <span th:each="label : ${entry.seasonLabels()}"
                          th:text="${label}"
                          class="season-tag"></span>
                </div>
            </div>
        </div>
    </div>
    <script>
        (function() {
            var select = document.getElementById('season-filter');
            var cards = document.querySelectorAll('.overview-card');
            select.addEventListener('change', function() {
                var slug = this.value;
                cards.forEach(function(card) {
                    if (!slug) {
                        card.style.display = '';
                    } else {
                        var seasons = card.getAttribute('data-seasons') || '';
                        card.style.display = seasons.split(' ').indexOf(slug) >= 0 ? '' : 'none';
                    }
                });
            });
        })();
    </script>
</section>
</body>
</html>
```

2. **Build `Map<Team, LinkedHashSet<Season>>`** (ordered by insertion = iteration order):
   ```java
   var sortedSeasons = productionSeasons.stream()
           .sorted(Comparator.comparing(Season::getYear).thenComparing(Season::getNumber))
           .toList();
   var teamToSeasons = new java.util.LinkedHashMap<org.ctc.domain.model.Team, java.util.LinkedHashSet<Season>>();
   for (var season : sortedSeasons) {
       for (var st : seasonTeamRepository.findBySeasonId(season.getId())) {
           var team = st.getTeam();
           if (!team.isSubTeam()) {  // D-01: filter sub-teams
               teamToSeasons.computeIfAbsent(team, k -> new java.util.LinkedHashSet<>()).add(season);
           }
       }
   }
   ```

3. **Build `List<TeamOverviewEntry>`** sorted alphabetically by shortName (Claude's Discretion):

   String assetsPath = "assets";  // Root-level page (Pitfall 5: assetsPath = "assets" for root pages)
   var teamEntries = teamToSeasons.entrySet().stream()
           .sorted(Comparator.comparing(e -> e.getKey().getShortName()))
           .map(e -> {
               var team = e.getKey();
               var seasons = new java.util.ArrayList<>(e.getValue());
               var latestSeason = seasons.getLast();  // most recent = last after sorted insertion
               String profileUrl = "season/" + slugify(latestSeason.getDisplayLabel())
                       + "/team/" + slugify(team.getShortName()) + ".html";
               String logoRelPath = copyLogoToAssets(team.getLogoUrl(), outPath, assetsPath);

                       team.getShortName(),
