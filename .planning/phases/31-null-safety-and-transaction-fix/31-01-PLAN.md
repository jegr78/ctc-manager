    GREEN PHASE: Refactor `executeImport()` in CsvImportService.java to two-phase structure:

    Phase 1 — Validate ALL team pairs (no writes):
    ```java
    // Phase 1: Validate all team pairs before any writes
    Map<String, Team[]> resolvedTeams = new LinkedHashMap<>();
    for (var entry : byTeamPair.entrySet()) {
        var teamParts = entry.getKey().split("\\|");
        var homeTeam = findTeamFlexible(teamParts[0], seasonTeams);
        var awayTeam = teamParts.length > 1 ? findTeamFlexible(teamParts[1], seasonTeams) : null;

        if (homeTeam == null) {
            result.addError("Team not found: " + teamParts[0]);
        }
        if (awayTeam == null && teamParts.length > 1) {
            result.addError("Team not found: " + teamParts[1]);
        }
        if (homeTeam != null) {
            var effectiveAwayTeam = awayTeam != null ? awayTeam : homeTeam;
            if (!overwriteExisting && matchRepository.existsByMatchdayIdAndHomeTeamIdAndAwayTeamId(
                    matchday.getId(), homeTeam.getId(), effectiveAwayTeam.getId())) {
                result.addError("Match already exists: " + homeTeam.getShortName() +
                        " vs " + effectiveAwayTeam.getShortName() + " on " + matchday.getLabel());
            }
            resolvedTeams.put(entry.getKey(), new Team[]{homeTeam, effectiveAwayTeam});
        }
    }

    if (result.hasErrors()) {
        return result;  // Abort — no writes happened (per D-02)
    }
    ```

    Phase 2 — Import (only reached if validation passes). Iterate `byTeamPair.entrySet()` again, using `resolvedTeams` to get pre-resolved teams. Remove the team resolution and duplicate check from the import loop (they are now in Phase 1). Keep the overwrite-delete logic in Phase 2 (only executes when overwriteExisting=true, which passes validation).

    IMPORTANT: Keep `@Transactional` on `executeImport()` per D-04. Do NOT add try-catch that swallows exceptions in Phase 2.

    Run tests — all must PASS (GREEN). Then run `./mvnw test -Dtest=CsvImportServiceTest` to verify all existing tests still pass.
