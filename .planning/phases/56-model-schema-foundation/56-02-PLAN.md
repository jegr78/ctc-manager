| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-56-03 | Tampering | Entity field values (e.g. crafted PhaseType) | mitigate | `@Enumerated(EnumType.STRING)` + typed `PhaseType`/`PhaseLayout`/`SeasonFormat` enums prevent invalid values. | T-56-05 | Information Disclosure | `@ToString` recursion exposing lazy collections | mitigate | All three entities declare `@ToString(exclude = {…})` listing every `@ManyToOne` parent and every `@OneToMany` collection (per CONVENTIONS.md §"Lombok Usage"). Prevents lazy-init triggers and cyclic toString output that could appear in logs. |
| T-56-06 | Denial of Service | Lazy-load N+1 on collection access | accept | OSIV is enabled; templates may load lazily. Performance optimisation via `@EntityGraph` is deferred to Phase 58 when service queries actually consume these collections (D-06). No new high-traffic path in Phase 56. |
