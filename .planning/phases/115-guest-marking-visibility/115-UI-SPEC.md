## Color

All tokens extracted from existing `:root` in `admin.css` and `site/css/style.css`.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0a0a0a` (`--bg`) | Page background |
| Secondary (30%) | `#141414` (`--bg-card`) | Cards, sidebar, nav; `#111` on site |
| Accent / Interactive (10%) | `#4fc3f7` (`--accent`) | Focus rings, active sidebar, border-focus, existing interactive elements |
| **Guest marker accent** | `#f59e0b` (`--guest`) | **Reserved exclusively for guest driver markers** ‚Äî glyph color + sub-label tint. New token, not a repurpose of `--accent`. |
| Destructive | `#d32f2f` (`--danger`) | Destructive actions only |
| Success | `#2e7d32` (`--success`) | Success states, active badges |

**`--guest: #f59e0b` rationale:**
- Amber separates cleanly from the existing blue (`--accent: #4fc3f7`) in all major color-vision-deficiency simulations (Deuteranopia, Protanopia, Tritanopia).
- Contrast ratio of `#f59e0b` on `#0a0a0a` = 7.9:1 (exceeds WCAG AA 4.5:1 and AA-large 3:1).
- Does not conflict with `--success` (green) or `--danger` (red) semantic colors.
- Icon (‚òÖ) is the primary meaning carrier; amber is reinforcement only (D-03).

**Accent reserved for (complete list):**
- `--accent` (#4fc3f7): focus ring (`.btn:focus-visible`), active sidebar border, `border-focus`, checkbox `accent-color`, existing interactive accents.
- `--guest` (#f59e0b): guest driver marker glyph (`‚òÖ`), guest sub-label on driver-profile, `isGuest` row tint in graphics.

## Guest Marker Specification

### Glyph

Unicode `‚òÖ` (U+2605, BLACK STAR) via `&#x2605;`.

**Rationale:** Language-neutral; intuitively signals "special/guest status"; no external library required; consistent with the existing `&#x25CF;` / `&#x25CB;` badge pattern in `season-detail.html`; stands alone without legend (D-09).

### CSS Classes (new, to append to `admin.css` and mirror into graphic templates)

```css
/* Guest marker ‚Äî append after existing .badge rules in admin.css */

--guest: #f59e0b;  /* append to :root block */

.guest-marker {
    color: var(--guest);
    font-size: 12px;
    margin-right: 4px;
    vertical-align: middle;
    line-height: 1;
}

.guest-label {
    font-size: 12px;
    font-weight: 400;
    color: var(--text-dim);
    margin-left: 4px;
}
```

For **graphic render templates** (inline CSS since they do not load admin.css):

```css
/* Inline in <style> block of each render template */
.guest-marker { color: #f59e0b; font-size: 0.85em; margin-right: 4px; }
```

### Shared Thymeleaf Fragment

Path: `src/main/resources/templates/admin/fragments/guest-marker.html`

```html
<!-- Usage: th:replace="~{admin/fragments/guest-marker :: guestMarker(${row.guest})}" -->
<th:block th:fragment="guestMarker(isGuest)">
    <span class="guest-marker" th:if="${isGuest}" aria-label="Guest driver">&#x2605;</span>
</th:block>
```

The `aria-label="Guest driver"` ensures screen readers announce guest status (icon alone is insufficient for accessibility).

### Surface-Specific Application

| Surface | Where marker appears | Fielding team shown? |
|---------|---------------------|---------------------|
| Graphics (`results-render`, `provisional-scores-render`, `lineup-render`) | Before driver name in `.driver-name` / `.col-driver` cell | No (binary only, D-04) |
| Admin `race-detail.html` + `matchday-detail.html` (lineup + result rows) | Before driver name; optionally: sub-team name in tooltip or extra `<span class="guest-label">` (D-08) | Optional ‚Äî actual sub-team name (D-11) |
| `site/driver-ranking.html` + `site/alltime-driver-ranking.html` | Before `psnId` in ranking row (when `hasGuestAppearance` is true on the ranking row) | No (binary only, D-04) |
| `site/driver-profile.html` (per-race result row) | Before driver name / in race row; plus inline sub-label "as guest for \<SubTeamName\>" | Yes ‚Äî required (MARK-06, D-10); show actual sub-team name (D-11) |

### Driver-Profile Sub-Label Copy

Format: `as guest for <SubTeamName>` ‚Äî lowercase "as guest for", then the actual sub-team short name from `RaceLineup`.

Example rendered: `‚òÖ Driver Name ‚Äî as guest for ALF-B`

Rendered as:
```html
<span class="guest-marker">&#x2605;</span>
<span class="guest-label" th:if="${row.guest}">as guest for <span th:text="${row.fieldingTeamName}"></span></span>
```
