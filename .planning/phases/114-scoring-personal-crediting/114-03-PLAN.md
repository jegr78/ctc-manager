<objective>
Extend `DriverProfilePageGenerator` so a **pure guest** — a driver who appears only in a `RaceLineup` for a season and has no `SeasonDriver` row — gets a public driver-profile page for that season (D-05). Scope fence (D-05 LOCKED): Phase 114 delivers the **data/page-existence hook ONLY**. NO visual guest marking, badge, `*`, CSS, or template change — that is Phase 115 (MARK-06). The second-pass page is structurally identical to the SeasonDriver-pass page; only the team source differs.

Purpose: SCORE-02 requires guest points to be visible per-driver; a pure guest with no roster entry would otherwise have no profile page on the public site.
Output: Second-pass loop in `DriverProfilePageGenerator.generate`; a sitegen test asserting the pure-guest page exists.
</objective>
