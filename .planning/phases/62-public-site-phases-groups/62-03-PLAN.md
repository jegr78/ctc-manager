`generate` algorithm:

```java
public void generate(GenerationContext ctx, GenerationResult result) throws IOException {
    List<SeasonPhase> allPhases = seasonPhaseService.findAllPhases(ctx.season().getId());
    boolean showPhaseTabs = allPhases.size() >= 2;
    String seasonSlug = siteSlugger.slugify(ctx.season().getDisplayLabel());
    Path dir = ctx.outPath().resolve("season").resolve(seasonSlug);
    Files.createDirectories(dir);

    List<UUID> phaseIds = allPhases.stream().map(SeasonPhase::getId).toList();

    // Build phase tabs once (active flag flips per variant)
    // First tab is always "All Phases" -> driver-ranking.html
    // Subsequent tabs are per-phase, including PLAYOFF (which DOES get its own driver-ranking-playoff.html when data exists)

    // 1) Legacy driver-ranking.html — cross-phase aggregated (UNCHANGED data source per D-11 SC4-clean)
    var legacyRanking = driverRankingService.aggregateAcrossPhases(phaseIds, ctx.season().getId());
    writeRankingVariant(ctx, dir, "driver-ranking.html", legacyRanking,
            showPhaseTabs,
            buildDriverPhaseTabs(allPhases, /*activePhaseId*/ null));
    result.incrementPages();

    // 2) Per-phase variants
    for (SeasonPhase p : allPhases) {
        var phaseRanking = driverRankingService.calculateRankingForPhase(p.getId());
        if (phaseRanking.isEmpty() && p.getPhaseType() == PhaseType.PLAYOFF) {
            // PLAYOFF with no driver data: skip variant; tab in row falls through to next variant on click — but since variant absent, the tab simply has no destination.
            // Codified rule: if PLAYOFF has no driver data, do NOT generate driver-ranking-playoff.html and OMIT the PLAYOFF entry from the tab list.
            continue;
        }
        String phaseSlug = p.getPhaseType().name().toLowerCase(Locale.ENGLISH);
        writeRankingVariant(ctx, dir, "driver-ranking-" + phaseSlug + ".html", phaseRanking,
                showPhaseTabs,
                buildDriverPhaseTabs(allPhases, p.getId()));
        result.incrementPages();
    }
}

private List<PhaseTabView> buildDriverPhaseTabs(List<SeasonPhase> allPhases, UUID activePhaseId) {
    List<PhaseTabView> tabs = new ArrayList<>();
    // First entry: "All Phases" -> driver-ranking.html, active when no specific phase is active
    tabs.add(new PhaseTabView("All Phases", "driver-ranking.html", activePhaseId == null, "main-driver-ranking"));
    // Per-phase tabs
    for (SeasonPhase p : allPhases) {
        // Skip PLAYOFF in tab row only if its variant won't be generated (per the "no driver data" rule above)
        // Pre-check by calling calculateRankingForPhase again:
        if (p.getPhaseType() == PhaseType.PLAYOFF
                && driverRankingService.calculateRankingForPhase(p.getId()).isEmpty()) {
            continue;
        }
        String label = (p.getLabel() != null && !p.getLabel().isBlank())
                ? p.getLabel()
                : capitalize(p.getPhaseType().name().toLowerCase(Locale.ENGLISH));
        String href = "driver-ranking-" + p.getPhaseType().name().toLowerCase(Locale.ENGLISH) + ".html";
        boolean active = activePhaseId != null && activePhaseId.equals(p.getId());
        tabs.add(new PhaseTabView(label, href, active, "main-driver-ranking"));
    }
    return tabs;
}

private void writeRankingVariant(GenerationContext ctx, Path dir, String filename,
                                  List<DriverRanking> ranking, boolean showPhaseTabs,
                                  List<PhaseTabView> phaseTabs) throws IOException {
    var model = new Context(Locale.ENGLISH);
    model.setVariable("season", ctx.season());
    model.setVariable("driverRanking", ranking);
    var driverSlugMap = new java.util.HashMap<UUID, String>();
    for (var r : ranking) {
        driverSlugMap.put(r.getDriver().getId(), "driver/" + siteSlugger.slugify(r.getDriver().getPsnId()) + ".html");
    }
    model.setVariable("driverSlugMap", driverSlugMap);
    model.setVariable("currentPage", "driver-ranking");
    model.setVariable("seasonSlug", siteSlugger.slugify(ctx.season().getDisplayLabel()));
    model.setVariable("seasonName", ctx.season().getName());
    model.setVariable("hasPlayoff", ctx.hasPlayoff());
    model.setVariable("playoffSeasonSlug", ctx.playoffSeasonSlug());
    model.setVariable("breadcrumbCurrent", "Driver Ranking");
    model.setVariable("showPhaseTabs", showPhaseTabs);
    model.setVariable("phaseTabs", phaseTabs);
    // No groupTabs — driver-ranking has no per-group variants
    model.setVariable("showGroupTabs", false);
    model.setVariable("groupTabs", List.of());
    templateWriter.write("site/driver-ranking", model, dir.resolve(filename),
            ctx.activeSeasonSlug(), ctx.activeSeasonName());
}
```

For real-world data (1-2 phases per season) this is negligible. If it becomes a bottleneck, cache the result in a `Map<UUID, Boolean>` per `generate(...)` invocation.
