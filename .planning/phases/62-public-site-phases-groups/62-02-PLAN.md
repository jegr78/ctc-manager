`generateIndex` algorithm:

```java
public void generateIndex(GenerationContext ctx, GenerationResult result) throws IOException {
    List<SeasonPhase> allPhases = seasonPhaseService.findAllPhases(ctx.season().getId());
    boolean showPhaseTabs = allPhases.size() >= 2;
    SeasonPhase regularPhase = allPhases.stream()
            .filter(p -> p.getPhaseType() == PhaseType.REGULAR)
            .findFirst().orElseThrow();

    String seasonSlug = siteSlugger.slugify(ctx.season().getDisplayLabel());
    Path dir = ctx.outPath().resolve("season").resolve(seasonSlug);
    Files.createDirectories(dir);

    // 1) Legacy matchdays.html — REGULAR-only matchdays (Open Question 2 locked)
    writeIndexVariant(ctx, dir, "matchdays.html",
            matchdayRepository.findByPhaseIdOrderBySortIndexAsc(regularPhase.getId()),
            allPhases, regularPhase, /*currentGroupId*/ null,
            showPhaseTabs, regularPhase.getLayout() == PhaseLayout.GROUPS,
            buildPhaseTabsForMatchdays(allPhases, /*currentPhaseType*/ regularPhase.getPhaseType(), /*isLegacyView*/ true),
            regularPhase.getLayout() == PhaseLayout.GROUPS
                    ? buildGroupTabsForMatchdays(regularPhase, "matchdays", null)
                    : List.of());
    result.incrementPages();

    // 2) Per-phase variants (skip PLAYOFF per D-08)
    for (SeasonPhase p : allPhases) {
        if (p.getPhaseType() == PhaseType.PLAYOFF) continue;
        String phaseSlug = p.getPhaseType().name().toLowerCase(Locale.ENGLISH);
        boolean isGroups = p.getLayout() == PhaseLayout.GROUPS;
        writeIndexVariant(ctx, dir, "matchdays-" + phaseSlug + ".html",
                matchdayRepository.findByPhaseIdOrderBySortIndexAsc(p.getId()),
                allPhases, p, /*currentGroupId*/ null,
                showPhaseTabs, isGroups,
                buildPhaseTabsForMatchdays(allPhases, p.getPhaseType(), false),
                isGroups ? buildGroupTabsForMatchdays(p, "matchdays-" + phaseSlug, null) : List.of());
        result.incrementPages();

        if (isGroups) {
            for (SeasonPhaseGroup g : seasonPhaseGroupRepository.findByPhaseIdOrderBySortIndex(p.getId())) {
                String groupSlug = siteSlugger.slugify(g.getName());
                writeIndexVariant(ctx, dir,
                        "matchdays-" + phaseSlug + "-group-" + groupSlug + ".html",
                        matchdayRepository.findByPhaseIdAndGroupIdOrderBySortIndexAsc(p.getId(), g.getId()),
                        allPhases, p, g.getId(),
                        showPhaseTabs, true,
                        buildPhaseTabsForMatchdays(allPhases, p.getPhaseType(), false),
                        buildGroupTabsForMatchdays(p, "matchdays-" + phaseSlug, g.getId()));
                result.incrementPages();
            }
        }
    }
}
```

Helper `writeIndexVariant` — encapsulates the Thymeleaf model setup (lifted from current `generateMatchdayIndex` body):

```java
private void writeIndexVariant(GenerationContext ctx, Path dir, String filename,
                                List<Matchday> matchdays,
                                List<SeasonPhase> allPhases, SeasonPhase currentPhase, UUID currentGroupId,
                                boolean showPhaseTabs, boolean showGroupTabs,
                                List<PhaseTabView> phaseTabs, List<GroupSubTabView> groupTabs) throws IOException {
    var model = new Context(Locale.ENGLISH);
    model.setVariable("season", ctx.season());
    model.setVariable("matchdays", matchdays);
    var matchdayLinkMap = new java.util.LinkedHashMap<UUID, String>();
    for (var md : matchdays) {
        matchdayLinkMap.put(md.getId(), "matchday/" + siteSlugger.slugify(md.getLabel()) + ".html");
    }
    model.setVariable("matchdayLinkMap", matchdayLinkMap);
    model.setVariable("currentPage", "matchdays");  // D-09 — stays coarse
    model.setVariable("seasonSlug", siteSlugger.slugify(ctx.season().getDisplayLabel()));
    model.setVariable("seasonName", ctx.season().getName());
    model.setVariable("hasPlayoff", ctx.hasPlayoff());
    model.setVariable("playoffSeasonSlug", ctx.playoffSeasonSlug());
    model.setVariable("breadcrumbCurrent", "Matchdays");
    model.setVariable("showPhaseTabs", showPhaseTabs);
    model.setVariable("showGroupTabs", showGroupTabs);
    model.setVariable("phaseTabs", phaseTabs);
    model.setVariable("groupTabs", groupTabs);
    templateWriter.write("site/matchdays", model, dir.resolve(filename),
            ctx.activeSeasonSlug(), ctx.activeSeasonName());
}
```
