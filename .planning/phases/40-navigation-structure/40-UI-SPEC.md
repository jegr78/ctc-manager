## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gap, `margin-bottom` on hero-label, breadcrumb separator margin |
| sm | 8px | Nav link padding-y, subnav gap between pills, `margin-bottom` on hero-sub |
| md | 16px | Default cell padding-x in tables, match-results margin-top |
| lg | 24px | Team header gap, card padding, footer padding-top |
| xl | 32px | Nav/subnav/main horizontal padding (desktop), section margin-bottom |
| 2xl | 48px | Hero padding, footer margin-top |
| 3xl | 64px | Not used by Phase 40 elements |

Exceptions:
- Subnav height: 44px (between sm and md rows — intentionally smaller than top-nav 60px to establish visual hierarchy)
- Nav top-nav height: 60px (existing, do not change)
- Mobile nav/subnav horizontal padding collapses to 16px at 768px breakpoint

Source: `style.css` existing patterns; subnav proportions per RESEARCH.md CSS Design Reference.

## Component Inventory

### 1. Subnav Bar (NEW — CONT-05)

**CSS class:** `.subnav` wrapping `.subnav-inner` wrapping `.subnav-link` items

**Visual spec:**
- Container: `background: var(--bg-card)`, `border-bottom: 1px solid var(--border)`, `padding: 0 32px`
- Inner: `max-width: 1100px`, `margin: 0 auto`, `display: flex`, `gap: 4px`, `height: 44px`, `align-items: center`
- Link inactive: `color: var(--text-dim)`, `font-size: 12px`, `text-transform: uppercase`, `letter-spacing: 1px`, `padding: 8px 12px`, `border-radius: 4px`, `transition: all 0.2s`
- Link hover: `color: var(--white)`, `background: rgba(255,255,255,0.05)`
- Link active: `color: var(--accent)`, `background: rgba(79,195,247,0.1)` — applied via `.subnav-link.active`

**Items (in order):** Standings | Matchdays | Driver Ranking | Playoff

**Conditional rendering:** `th:if="${seasonSlug != null}"` — renders only when season context is present. Archive and index pages set `seasonSlug = null` and skip the subnav.

**Mobile (≤768px):** Horizontal scroll — `overflow-x: auto`, `-webkit-overflow-scrolling: touch`. `subnav-inner` height collapses to `auto` with `padding: 8px 0`. Links do NOT stack vertically (keep horizontal for tab-bar feel).

Source: D-02 from CONTEXT.md; CSS Design Reference from RESEARCH.md.

### 2. Active Nav State (NEW — UX-02)

**CSS class added to top-nav links:** `.nav-link-active`

**Visual spec:**
- Same treatment as subnav active: `color: var(--accent)`, `background: rgba(79,195,247,0.1)`
- Applied via `th:class="${currentPage == 'X'} ? 'nav-link-active' : ''"` on existing `.nav-links a` elements

**Top-nav active mapping:**

| `currentPage` value | Top-nav item highlighted |
|--------------------|--------------------------|
| `index` | Standings (→ index.html) |
| `driver-ranking` (active season's ranking) | Driver Ranking |
| `archive` | Archive |
| All other values | No top-nav item highlighted |

**Subnav active mapping:**

| `currentPage` value | Subnav item highlighted |
|--------------------|--------------------------|
| `standings` | Standings |
| `matchdays` | Matchdays |
| `driver-ranking` | Driver Ranking |
| `playoff` | Playoff |
| `team`, `driver`, null | No subnav item highlighted (season context still shows) |

Source: D-05, D-06, D-07 from CONTEXT.md; Pattern 3 from RESEARCH.md.

### 3. Breadcrumb (NEW — UX-03)

**CSS class:** `.breadcrumb` nav element containing `.breadcrumb-link`, `.breadcrumb-sep`, `.breadcrumb-current`

**Visual spec:**
- Container: `font-size: 12px`, `padding: 8px 32px`, `color: var(--text-muted)`, `max-width: 1100px`, `margin: 0 auto`
- Position: between subnav and `<main>` content — inside `layout.html`, outside the `.main` wrapper
- Breadcrumb link: `color: var(--text-dim)`, `text-decoration: none` — on hover: `color: var(--accent)`
- Separator `>`: literal `<span class="breadcrumb-sep" aria-hidden="true"> > </span>`, `margin: 0 8px`, `color: var(--text-muted)` — NOT via `th:text` (avoids HTML-entity escaping issue)
- Current page: `<span class="breadcrumb-current">`, `color: var(--text)` — plain text, not a link

**Three-level structure:**
```
Home > {Season Name} > {Page Title}
```
- "Home" links to `${rootPath}/index.html`
- Season Name links to `${rootPath}/season/${seasonSlug}/standings.html`
- Current page: plain text from `breadcrumbCurrent` variable

**Conditional rendering:** `th:if="${breadcrumbCurrent != null}"` — renders only when `breadcrumbCurrent` is set. Archive and index set `breadcrumbCurrent = null`.

**Mobile (≤768px):** `padding: 8px 16px` — matches nav mobile padding.

Source: D-08 through D-11 from CONTEXT.md; Pattern 4 and Pitfall 5 from RESEARCH.md.

---

### 4. Matchday Index Page (NEW — CONT-05)

**Template:** `src/main/resources/templates/site/matchdays.html`
**Output path:** `season/{slug}/matchdays.html`

**Visual spec:** Reuse existing table pattern — `.table-wrap` > `table` with standard `th`/`td` padding.

**Columns:**
- Matchday label (clickable link → individual matchday page) — renders as `.entity-link`
- No date column (field does not exist on `Matchday` entity — verified A3 in RESEARCH.md)

**Empty state:** "No matchdays recorded for this season." — plain text in a `<p class="text-dim">` inside `.section`. No icon, no CTA.

**Breadcrumb:** `breadcrumbCurrent = "Matchdays"`
**Active subnav:** `currentPage = "matchdays"`

Source: D-12, D-13, D-14 from CONTEXT.md; Pattern 5 from RESEARCH.md.

## Template Variable Contract

Every `generate*()` call for a season subpage MUST set these variables before calling `writeTemplate()`:

| Variable | Type | Season Pages | Archive / Index |
|----------|------|-------------|-----------------|
| `currentPage` | `String` | page identifier (see mapping above) | `null` |
| `seasonSlug` | `String` | slugified season display label | `null` |
| `seasonName` | `String` | season name (for breadcrumb) | `null` |
| `breadcrumbCurrent` | `String` | page title string | `null` |

Explicitly setting null variables for archive/index prevents Thymeleaf `TemplateInputException` when `th:if` guards evaluate the variable.

Source: RESEARCH.md Common Pitfall #2.

## Claude's Discretion Resolutions

| Item | Decision | Rationale |
|------|----------|-----------|
| Index page subnav (D-04) | Omit subnav on index page | Index is the global home; "Standings" in subnav would appear to navigate to the same page as the "Standings" top-nav link. Clean separation: index = home, season pages = seasonal context. |
| Subnav CSS dimensions | height: 44px, font-size: 12px, gap: 4px, padding: 8px 12px per link | Proportionally smaller than top-nav (60px / 13px) — establishes clear visual hierarchy without being illegible |
| Breadcrumb separator | Literal `<span aria-hidden="true"> > </span>` — not via `th:text` | Avoids HTML entity escaping in Jsoup test assertions (Pitfall 5) |
| Matchday date display | Label-only — no date column | `Matchday.matchDate` field does not exist (A3 verified); no database change in scope |
| Mobile subnav | Horizontal scroll (not vertical stack) | Maintains tab-bar UX familiar from mobile apps; vertical stack would break the pill design |
