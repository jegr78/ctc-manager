private void writeTemplate(String templateName, Context context, Path outputFile,
                            String activeSeasonSlug, String activeSeasonName) throws IOException {

    Path relativeAssets = outputFile.getParent().relativize(outRoot.resolve("assets"));

}

|----------|-------------|
| YAML config to SiteProperties | Developer-controlled config; not user input |
| SiteProperties to Thymeleaf template | Link URLs rendered via th:href; names escaped via th:text |
| Static HTML output to browser | Generated pages with external links |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-46-01 | Spoofing | External link URLs from config | accept | Links from application.yml controlled by developer; no user input |
| T-46-02 | Tampering | Link card content | mitigate | Thymeleaf `th:text` auto-escapes HTML; `th:href` renders URL as-is but config is developer-controlled |
| T-46-03 | Information Disclosure | External links open in new tab | mitigate | `rel="noopener"` attribute on all external `<a>` elements prevents tab-napping (per D-07) |
</threat_model>
