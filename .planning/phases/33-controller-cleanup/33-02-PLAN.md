<objective>
Fix SiteGeneratorService.toRaceView() to use RaceLineup as primary source of truth for driver-team assignment, with SeasonDriver as fallback (per D-05, D-07). This ensures generated site pages correctly attribute drivers to their sub-teams as recorded in race lineups.

Purpose: The site generator currently reads team assignment from SeasonDriver only, which does not reflect sub-team assignments recorded per-race in RaceLineup entries. This causes drivers to be attributed to the wrong team on generated site pages. The canonical RaceLineup-first pattern is already established in ScoringService.isDriverInTeam() and RaceFormDataService.toRaceData().
Output: Fixed toRaceView() method, new RaceLineupRepository injection, integration test proving RaceLineup-primary resolution.
