## Cross-Phase Integration (checker verdict: PASS)

Full E2E flow traces without interruption:

1. **Assign** (113) — admin selects any driver, fields for a team → `RaceLineupService.saveLineup` persists `RaceLineup(..., isGuest=true)` (`is_guest` V18 column).
2. **Record** (113) — `RaceFormDataService` auto-derives the guest into the results form → `RaceService.saveResults` saves `RaceResult` + calls `aggregateMatchScores`.
3. **Credit team** (114) — `ScoringService.isDriverInTeam` resolves via `RaceLineup.getTeam()` (fielding team) → guest points credit the fielding side (SCORE-01).
4. **Credit driver** (114) — `DriverRankingService.resolveAttributedTeam` home-first → fielding-team fallback; pure guests (no SeasonDriver) included via lineup backfill (SCORE-02); full-recompute aggregation is idempotent (SCORE-03).
5. **Mark** (115) — `RaceLineup.isGuest()` / `hasGuestAppearance` drive the star marker on all six MARK surfaces, consistently (single source of truth — CLAUDE.md "RaceLineup is Source of Truth").

13/13 cross-phase connections wired; 0 missing; 0 broken flows.
